1. configurar o Git 
git config --global user.name "mariaaraquell"
git config --global user.email "mariaeduardarb.senai@gmail.com"

2. inicializar o git 
git init

3. criar o repo no githug
nome do repo: Aula01

4. nessa etapa, é a etapa de desenvolvimento ou seja, alterações nos nossos arquivos. site, projetos, arquivos textos, entre outros 

5. após as conclusão do desenvolvimento, adicionar os arquivos ao repo:
git add .

6. comitar ao repositório com uma mensagem explicativa: git commit -m "mensagem entre parênteses"

7. associar o nosso repo local com o remoto 
git remote add origin LINK_DO_REPO
git branch -M main
git push -u origin main
